# Add Pull Request Template

## Summary
Added a standardized pull request template to guide contributors and ensure consistent PR submissions.

## Changes Made
- Created `.github/PULL_REQUEST_TEMPLATE.md` with comprehensive template including:
  - Description section
  - Type of change checkboxes
  - Testing instructions
  - Checklist for code quality
  - Screenshots section
  - Additional notes

## Template Content
The template includes sections for:
- Brief description of changes
- Categorization of change type (bug fix, new feature, etc.)
- Testing verification requirements
- Code quality checklist
- Screenshots for UI changes
- Additional context

## Verification
- File created at `.github/PULL_REQUEST_TEMPLATE.md`
- Template follows GitHub's standard format
- All sections are optional but encouraged for thorough PRs

This will help maintain high-quality contributions and make code reviews more efficient.
